This is the code of my first portfolio site. It was built with Vue 3.

## Live version
patrikvisloczki.com

## Run it locally

 1. Clone the repo
 `$ git clone https://github.com/natixco/portfolio.git`
2. Install the dependencies
`$ npm i`
3. Install @vue-cli globally
`$ npm i -g @vue-cli`
4. Run the site
`$ npm run serve`
